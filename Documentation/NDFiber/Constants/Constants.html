NDFiberSucceedToFinishedFiberException

Version: 1.0.0

extern NSString * NDFiberSucceedToFinishedFiberException;
Discussion

An NDFiber will throw this exception if you try yield to a fiber that has finished executing.

NDFiberWillExitNotification

Version: 1.0.0

extern NSString * NDFiberWillExitNotification;
Discussion

An NDFiber object posts this notification when it receives the exit message, before the thread exits. Observer methods invoked to receive this notification execute in the exiting thread, before it exits. The notification object is the exiting NDFiber object. This notification does not contain a userInfo dictionary.
